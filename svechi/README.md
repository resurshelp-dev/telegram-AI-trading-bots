# Svechi

Главный запуск только один:

- `launch_svechi_live.cmd`

Внутренний технический runner:

- `run_svechi_final.ps1`

Настройки:

- `svechi.env`
- `svechi.env.example`

Что осталось в корне:

- один launcher
- один внутренний runner
- код системы
- конфиг
- краткая документация

Старые launchers и альтернативные runners перенесены в:

- `legacy_launchers/`

Если нужен обычный запуск системы, запускай только:

- `launch_svechi_live.cmd`
