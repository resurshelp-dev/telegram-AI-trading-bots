# 3bar

Главный запуск только один:

- `launch_3bar_window.cmd`

Внутренний технический runner:

- `run_3bar_system.ps1`

Настройки:

- `.env`
- `.env.example`

Что осталось в корне:

- один launcher
- один внутренний runner
- код системы
- конфиг
- краткая документация

Старые launchers и вспомогательные раннеры перенесены в:

- `legacy_launchers/`

Если нужен обычный запуск системы, запускай только:

- `launch_3bar_window.cmd`
